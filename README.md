# iherb_project

## '목욕 & 퍼스널 케어' 제품 리스트 scraping
- i-herb에서 취급하는 제품 중 약 총 5500개에 달하는 **목욕 & 퍼스널 케어 제품**을 scrap
- 제품 리스트 페이지에서 보여지는 정보만 취급
- scrap 해온 정보를 저장한 DB 테이블 상세
